# OS CPU Scheduler

C systems project for exploring CPU scheduling behavior with workload files, tests, and a reproducible Make workflow.

## Focus

- Scheduling algorithm implementation
- Workload parsing and simulation
- Testable operating-systems behavior
- Small, inspectable C codebase for portfolio review

## Repository Structure

- `src/`: scheduler implementation
- `tests/`: project tests
- `scripts/`: helper scripts
- `Makefile`: build and test entrypoint
- `workload.txt`: sample workload input

## Build

```bash
make
```

Run the included tests or scripts from the repository root after building.
